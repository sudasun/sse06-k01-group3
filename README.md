# Gabage Talk (ガベージトーク)

## 使用方法

1. マイクに向かって手に持っているゴミを声に出す
  - 現在の対応品目
    - 紙コップ
    - バナナの皮
    - ティッシュ
    - 食品トレイ
    - お菓子のごみ 
2. 対応したゴミ箱の蓋がパカっと空きます
   - 燃えるゴミ
     - 紙コップ
     - バナナの皮
     - ティッシュ
   - プラスチック
     - 食品トレイ
     - お菓子のごみ
